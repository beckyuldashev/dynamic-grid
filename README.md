# Dynamic Responsive Grid
### Dynamic Responsive Grid
В данном проекте мы посмотрим, как же создать динамическую, отзывчивую сетку с помощью CSS.
Воспользуемся такими свойствами CSS как:
- aspect-ratio
- object-fit
- много Modern CSS.
